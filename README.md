Overview
The Contact Manager is a JavaFX-based desktop application that allows users to manage their contacts efficiently. It provides functionalities to add, update, delete, and search for contacts. The application maintains contacts in a sorted manner and supports input validation and error handling. Contacts are stored in a CSV file, ensuring data persistence between sessions.

Features
CRUD Operations: Create, read, update, and delete contacts.
Search Functionality: Search for contacts by name or phone number.
Data Persistence: Contacts are saved to and loaded from a CSV file.
Input Validation: Validates phone numbers and email addresses.
Error Handling: Displays user-friendly error messages for invalid operations.
Tech Stack
Language: Java
GUI Framework: JavaFX
Data Storage: CSV File
Prerequisites
Java Development Kit (JDK) 8 or newer
An IDE with support for Java and JavaFX (e.g., IntelliJ IDEA, Eclipse)
Setup and Running
Clone the repository or download the project files.
Open the project in your preferred IDE.
Ensure JavaFX library is configured properly in your IDE.
Run the ContactManager class to start the application.
